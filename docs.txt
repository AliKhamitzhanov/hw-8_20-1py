CREATE - это команда которая создает новую таблицу данных.

INSERT - это команда которая добавляет данные в таблицу.

SELECT - это команда которая получает данные из выбранной таблицы.
         Например: SELECT * FROM <table_name>

UPDATE - это команда которая обновляет данные таблиц.

DROP - команда которая удаляет string из таблицы или всю таблицу целиком или целоую базу данных.
